- [ ] Вопрос: Можно ли послать GET запрос с body. Что произойдет если да, если нет то почему ?
Послать запрос GET c боди можно, поскольку http это позволяет поскольку body есть во всех запросах. Но из рекомендаций HTTP следует, что вы можете отправить тело запроса с помощью GET, но это не должно иметь никакого значения. Если вы придаете этому значение, анализируя его на сервере и изменяя свой ответ на основе его содержимого, то вы игнорируете эту рекомендацию по HTTP. 
Я сам попробовал отправить тело на сервер развернутый на TomCat + Spring boot + java и все прошло без ошибок.
